# Public Audit Limits

This repo is intentionally narrow.

A public audit of this repository can establish:

- exact bounded replay for the committed endogenous visual form corpus
- parity between the Rust-native path and the in-repo Python reference path
- explicit non-claims for broad cognition and non-visual semantics

A public audit of this repository cannot establish:

- general cognition decoding
- broader decoding of non-visual mental content
- clinical efficacy
- claims outside the files committed here

If a statement requires any of those broader conclusions, it is out of scope for this release.
