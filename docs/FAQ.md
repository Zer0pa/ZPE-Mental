# FAQ

## What does this repo encode?

Bounded endogenous visual forms represented as geometric trajectories with explicit profile and symmetry metadata.

## What is out of scope?

Broad cognition, language planning, autobiographical memory, abstract reasoning, and any broader decoding claim outside endogenous visual forms.

## Why does the repo ship both Rust and Python paths?

The Rust path is the native fast path. The Python path stays in-repo as the public reference implementation used for parity checks.

## Is this a clinical tool?

No. This repo is not a diagnostic, therapeutic, or clinical product.

## Does this repo depend on a broader stack?

No. This repository stands on its own as one bounded encoding product.
